# Contributing to Conveyor

Thank you for your interest in contributing to Conveyor! We welcome contributions from everyone.

## Ways to Contribute
- **Bug Reports**: If you find a bug, please open an issue with a clear description and steps to reproduce.
- **Feature Requests**: If you have an idea for a new feature, open an issue to discuss it.
- **Code Contributions**: 
    1. Fork the repository.
    2. Create a new branch for your feature or fix.
    3. Ensure your code passes `cargo check`, `cargo fmt`, and `cargo clippy`.
    4. Submit a Pull Request.

## Development Setup
1. Install Rust: https://rustup.rs/
2. Clone your fork: `git clone https://github.com/yourusername/conveyor.git`
3. Run the project: `cargo run`
4. Run tests: `cargo test`

## Code Standards
- Follow idiomatic Rust patterns.
- Keep the TUI responsive and high-signal.
- Update the `README.md` if you add new features.

# Security Policy

## Reporting a Vulnerability

We take the security of Conveyor seriously. If you find a security vulnerability, please do not open a public issue. Instead, please report it through one of the following methods:

- Email: [sumant1122@gmail.com](mailto:sumant1122@gmail.com)

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Supported Versions

Currently, only the latest version of Conveyor is supported for security updates.
